# AI Career Mentor

AI Career Mentor is a web-based platform designed to help students explore career opportunities, generate personalized learning roadmaps, and access curated learning resources.

## Features

* AI-inspired Career Recommendations
* Personalized Learning Roadmaps
* Career Resource Suggestions
* Interactive Chat Assistant
* Dark Mode Support
* Progress Tracker
* Download Roadmap Feature
* Local Storage Support
* Responsive Design
* JSON-Based Dynamic Content

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* JSON
* Local Storage API

## How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Install the Live Server extension.
4. Right-click on index.html.
5. Select "Open with Live Server".
